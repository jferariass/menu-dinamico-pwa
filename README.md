# Menu Dinámico 🚀

Premium Digital Menu PWA and SaaS Platform.

## Features
- **Luxury Aesthetic**: Dark mode, glassmorphism, and high-end animations.
- **Dynamic Themes**: Real-time palette adjustment based on merchant branding.
- **SaaS Ready**: Multi-tenant architecture for restaurant owners.
- **Order Flow**: Integrated cart for Dine-In and Delivery.

## Tech Stack
- HTML5 / Vanilla CSS3 / JavaScript
- Firebase (Real-time DB & Auth)
- PWA (Service Workers & Manifest)
